## Overview
Built a comprehensive automated retail system to minimize checkout latency. Integrated RFID-based inventory tracking with a dynamic billing engine.

## Technical Details
- **Backend**: Robust **C pipeline** using custom structures for product/customer management and file-based persistence.
- **Real-time Sync**: Automated inventory deduction upon purchase.
- **Reporting**: Generated daily transaction logs for sales analysis.

## Key Metrics
- Reduced checkout time by **60%**.
- Scaled to support **~1000 customers/day** and simultaneous updates to 500+ products.
